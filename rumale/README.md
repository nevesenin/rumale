Rumale (**Ru**by **ma**chine **le**arning) is a machine learning library in Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rumale'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install rumale
